Nimiq Pop-Up Shop Setup

Hello! :)

It takes two steps to set up your shop: store the private key savely and customize your shop's configuration.

Your private key

Copy the text from the box below and store it somewhere safe, e.g. in an encrypted ZIP file. If somebody gets this text (your private key), this person has access to your shop and all the orders.

Don't share, store safely.

Your configuration

  1. Edit name and address
  2. Add a logo if you like (if not, remove the line)
  3. Set live: true once you tested everything and you're ready for your first customers
  4. Don't touch anything below (id and publicKey)
  5. Copy/paste the entire thing into a file and named nimiq-popup-shop-configuration.json and put it the root of your shop. Note: if you want to put it in another place, make sure to add config="[file location]" to <nimiq-shop>.
All done, go to the shop backend
