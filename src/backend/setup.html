Nimiq Pop-Up Shop Setup

Hello! :)

There are two steps below to set-up your Nimiq Pop-Up Shop.

Your private key

Copy the text from the box below and store it somewhere safe, e.g. in an encrypted ZIP file. If somebody gets this text (your private key), this person has access to your shop and all the orders.

Don't share, store safely.

Your configuration

  1. for Edit name and address
  2. Add a logo if you like (if not, remove the line)
  3. Set live: true once you tested everything and you're ready for your first customers
  4. Don't touch anything below (id and publicKey)
  5. Copy/paste the entire thing into a file and named nimiq-popup-shop-configuration.json and put it the root of your shop.
